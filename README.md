## Description

Monorepo for Babysitters app built with Nest-js, React and React Native.

## Prerequisites

[MongoDB](https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/)

## Installation

```bash
$ npm install
```

## Running the db

```bash
# mongodb development macOS
$ brew services start mongodb-community@4.2
```

## Running the app

```bash
# developmen
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
